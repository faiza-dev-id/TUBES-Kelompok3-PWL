# 🎓 Sistem Manajemen Magang Mahasiswa

Nama WEBSITE : SIGMA (Sistem Informasi Magang Mahasiswa)

---

## 📌 Deskripsi Sistem

SIGMA adalah aplikasi berbasis web yang digunakan untuk mengelola proses magang mahasiswa mulai dari pendaftaran, seleksi oleh mitra, monitoring kegiatan magang, hingga penilaian dan laporan akhir secara terintegrasi.

---

## 🌐 Daftar Halaman Website

Berikut halaman-halaman utama pada sistem:

* Home
* Login
* Register
* Dashboard Admin
* Dashboard Mahasiswa
* Dashboard Mitra
* Daftar Lowongan Magang
* Detail Lowongan Magang
* Halaman Apply Magang
* Halaman Kegiatan Magang
* Halaman Penilaian Magang
* Halaman Laporan Magang
* Dashboard Analytics

---

## 👥 Fitur Berdasarkan Pengguna

### 🎓 Mahasiswa

Mahasiswa dapat menggunakan sistem untuk:

* Melihat daftar lowongan magang
* Mengajukan lamaran magang
* Mengisi log kegiatan magang
* Melihat hasil penilaian magang

### 🏢 Mitra

Mitra dapat menggunakan sistem untuk:

* Melihat lamaran mahasiswa
* Menerima atau menolak lamaran
* Memantau log kegiatan mahasiswa
* Memberikan penilaian magang

### 🛠️ Admin

Admin bertugas untuk:

* Mengelola data sistem
* Membuat akun mitra
* Memastikan proses magang berjalan dengan baik

### 📋 Ketua Prodi / Sekretaris Prodi

Bertugas untuk:

* Memantau kegiatan magang mahasiswa
* Melihat laporan hasil magang

---

## 🔄 Alur Sistem

Proses utama dalam sistem magang adalah:

1. Mahasiswa login ke sistem
2. Mahasiswa melihat daftar lowongan
3. Mahasiswa melakukan apply magang
4. Mitra memeriksa lamaran mahasiswa
5. Jika diterima, mahasiswa mulai kegiatan magang
6. Mahasiswa mengisi log kegiatan
7. Mitra memantau kegiatan
8. Mitra memberikan penilaian
9. Sistem menampilkan hasil magang

---

## 📊 Dashboard Analytics

Dashboard analytics menampilkan informasi penting terkait pelaksanaan magang, seperti:

* Jumlah mahasiswa yang mengikuti magang
* Jumlah lowongan magang yang tersedia
* Status lamaran mahasiswa *(pending, diterima, ditolak)*
* Perusahaan dengan jumlah mahasiswa magang terbanyak
* Rata-rata nilai mahasiswa

### Grafik yang ditampilkan:

* Grafik jumlah mahasiswa magang per perusahaan
* Grafik status lamaran magang
* Grafik rata-rata nilai mahasiswa

---

## ✅ Aturan Tambahan Sistem dari Dosen

### 1. Admin Membuat Akun Mitra

Akun mitra hanya dapat dibuat oleh **admin**, sehingga perusahaan yang terdaftar dapat dipastikan merupakan perusahaan yang valid dan terpercaya.

### 2. Alasan Penolakan Lamaran

Jika lamaran mahasiswa ditolak, maka pihak **mitra wajib memberikan alasan penolakan**, agar mahasiswa mengetahui alasan penolakan dan dapat melakukan perbaikan pada lamaran berikutnya juga agar prodi tahu apa yang kurang dari mahasiswa dari jurusannya agar bisa di evaluasi.
